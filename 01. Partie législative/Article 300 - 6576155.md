# Article 300

Si les accusés ne se concertent pas pour récuser, le sort règle entre eux le rang dans lequel ils font les récusations. Dans
ce cas, les jurés récusés par un seul, et dans cet ordre, le sont pour tous jusqu'à ce que le nombre des récusations soit
épuisé.
